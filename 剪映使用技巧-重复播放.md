# 剪映使用技巧

## 音频重复播放

### 场景
需要让某个音频片段在时间线上重复播放多次

### 方法

#### 方法1：复制粘贴（最快）
1. 选中音频片段
2. 按 `Cmd + C` 复制
3. 按 `Cmd + V` 粘贴
4. 继续按 `Cmd + V` 多次粘贴直到达到需要的重复次数

#### 方法2：Option + 拖拽
1. 按住 `Option` 键
2. 拖拽音频片段到新位置
3. 释放鼠标创建副本
4. 重复操作创建多个副本

#### 方法3：翻倍法（快速填满时间线）
1. 先复制一个片段
2. 选中两个片段一起复制
3. 粘贴（每次翻倍，快速填充）

### 快速预览循环
- 按 `I` 键设置入点
- 按 `O` 键设置出点
- 播放时会在该区间循环
